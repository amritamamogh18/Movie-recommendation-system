# 🎬 CineMatch - Movie Recommendation Web App

CineMatch is a personalized movie recommendation platform that helps users discover movies based on genre, mood, release year, and ratings. It’s your digital movie buddy built with love and a bit of AI magic.

![CineMatch Screenshot](./screenshots/cinematch_home.png)

---

## 🚀 Features

- 🎭 Filter by genre: Action, Drama, Comedy, Romance, Thriller, etc.
- 😌 Mood-based filtering: Happy, Relaxed, Thoughtful, Excited
- 📅 Year slider to filter by movie release period
- ⭐ Rating-based sorting
- 🔍 Search bar for direct lookups
- 👤 User login/logout system
- 🧠 Smart recommendations with support for hybrid or ML models (if integrated)
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js / Express (optional if server is used)
- **Data Source:** TMDb API / custom dataset
- **Authentication:** Firebase Auth or mock (based on setup)
- **State Management:** Redux / Context API
- **Hosting:** Vercel / Netlify / Render (choose your flavor)
